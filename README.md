# Novex AI Service Desk Engineer

Novex is an AI-powered Tier-1 Service Desk Engineer designed to resolve
high-volume Microsoft 365 support issues using deterministic state machines,
confidence-based escalation, and enterprise-safe automation.

## MVP Scope
- Password & account access issues
- Microsoft Teams client issues
- Microsoft Outlook client issues
- Clean escalation to L2 with full context

## Non-goals (MVP)
- Voice support
- RMM automation
- Multi-tenant support
- Mac/Linux environments

## Architecture Principles
- Deterministic state machines (not free-form chatbots)
- Least-privilege automation
- Explicit consent for sensitive actions
- Audit-first design
